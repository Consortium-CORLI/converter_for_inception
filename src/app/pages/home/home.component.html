Accueil

Cet utilitaire est un prototype qui doit permettre de préparer des documents XML pour être importés dans l'outil INCEpTION. Ceci implique de transformer vos données dans le format UIMA CAS XMI avec un seul document par fichier (split).

Marche à suivre :
  1. Charger votre document XML (TEI par exemple)
  2. Le document sera analysé (SAX parser) et présenté dans une table où les colonnes représentent un niveau de profondeur dans l'arborescence du document XML
  3. Il vous est alors demandé d'identifier dans ce document :
    1. La balise qui permet de distinguer chaque document, si votre document comporte une liste de documents
    2. [optionnel] : la balise qui représente la notion de paragraphe
    3. [optionnel] : la balise qui représente la notion de phrase
    4. les annotations pré-éxistantes qui vous souhaitez retrouver dans INCEpTION

L'identification des éléments précités se fait par clic-sélection d'une cellule (balise) dans la table puis en cliquant sur les boutons d'actions correspondant à ce que l'on veut identifier. Le bouton Collecter permet de collecter toutes les balises du document. À l'utilisateur d'éditer le type de la balise dans la table (doc càd frontière de document, annotation).

  • Objectif 1 : Convertir le fichier en entrée vers des fichiers CAS XMI.
  • Objectif 2 : Générer un script Python permettant de réaliser cette conversion
  • Objectif 3 (plus lointain) : Générer un script Python permettant de rétro-convertir l'export du projet INCEPTION vers le format original.
